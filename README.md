## Getting Started
To clone:

```
git clone --recursive https://github.com/rpuhalovich/OpenGLPathfinder.git
cd OpenGLPathfinder
```

Build for Windows:

```cmake -S . -B build -G"Visual Studio 16 2019"``` or ```scripts\build_visual_studio.bat```

Build for Mac:

```cmake -S . -B build -GXcode``` or ```./scripts/build_xcode.sh```
